# 👋 Welcome

Welcome to my Homelab Documentation repository.

This repository contains installation guides, configuration examples, troubleshooting notes, and infrastructure documentation used across my homelab environment.

The goal of this repository is simple:

- Keep documentation organized
- Make deployments repeatable
- Preserve knowledge for future use
- Build a personal infrastructure knowledge base
